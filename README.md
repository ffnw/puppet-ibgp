## Description

Manages iBGP route exchange.

To do so this module creates bird configuration files.

## Setup

### Setup Requirements

* puppet modules
  * ffnw-bird

### Beginning with ibgp

```puppet
class { 'ibgp':
  local_as => 65536,
  source   => '1.2.3.4',
  source6  => 'fc00::1',
}
```

## Usage

```puppet
class { 'ibgp':
  local_as => 65536,
  source   => '1.2.3.4',
  source6  => 'fc00::1',
}

ibgp::peer { 'gw02':
  neighbor  => '1.2.3.5',
  neighbor6 => 'fc00::2',
}
```
